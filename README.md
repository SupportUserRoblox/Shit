# My Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, and JavaScript.

## Features

- ✅ Responsive design (mobile, tablet, desktop)
- ✅ Smooth animations and transitions
- ✅ Mobile navigation menu
- ✅ Modern gradient design
- ✅ Contact form
- ✅ Project showcase
- ✅ Sticky header navigation

## Sections

1. **Header** - Navigation menu with logo
2. **Hero Section** - Welcome banner with call-to-action
3. **About Section** - Introduction and skills
4. **Projects Section** - Portfolio of work
5. **Contact Section** - Contact form
6. **Footer** - Social links and copyright

## File Structure

```
.
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
└── README.md
```

## How to Use

1. Clone this repository
2. Open `index.html` in your browser
3. Customize the content with your own information
4. Deploy to GitHub Pages

## Customization

- Edit `index.html` to change content
- Modify `css/style.css` to change colors and styling
- Update `js/script.js` to add functionality
